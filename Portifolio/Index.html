Batalha Espacial

O "Batalha Espacial" é um jogo rápido e divertido, ideal para dispositivos móveis. O foco foi criar uma experiência de jogo fluida, com sistema de pontuação e aumento progressivo da dificuldade. Desenvolvi este projeto para aprimorar minhas habilidades em detecção de colisão e gerenciamento de estado do jogo.

Neste projeto, utilizei uma abordagem que me permitiu aprender sobre a otimização para performance em plataformas móveis e a criação de interfaces de usuário responsivas para diferentes tamanhos de tela.

Corrida do Robô (Projeto Scratch)

Este é um projeto interativo de programação visual desenvolvido no **Scratch**. Ele demonstra a criação de um jogo simples de corrida onde o jogador controla um robô que deve desviar de obstáculos.

O projeto foi fundamental para solidificar meu entendimento de **lógica de programação**, como loops, condicionais e detecção de eventos, de uma forma visual e divertida. É um excelente exemplo de como a programação pode ser acessível e criativa para iniciantes.
